SpringBoot2 路由配置示例

基本路由示例

  • 基本GET请求 - /basic/hello
  • 带参数GET请求 - /basic/param?name=测试用户
  • 路径变量 - /basic/path/123

REST风格API示例

  • 获取所有用户 - GET /api/users
  • 获取单个用户 - GET /api/users/1
  • 创建用户 - POST /api/users (需使用Postman等工具测试)
  • 更新用户 - PUT /api/users/1 (需使用Postman等工具测试)
  • 删除用户 - DELETE /api/users/1 (需使用Postman等工具测试)

高级路由示例

  • 请求头信息 - /advanced/headers
  • Cookie信息 - /advanced/cookies
  • Session示例 - /advanced/session

静态资源示例

  • 图片资源 - /static/images/logo.png
  • JS资源 - /static/js/main.js
  • CSS资源 - /static/css/main.css

Actuator端点

  • Actuator主页 - /actuator
  • 健康检查 - /actuator/health
  • 应用信息 - /actuator/info
